#Scala Tutorial

This is a simple Scala tutorial aimed at beginners, covering:
 
 * SBT
 * HTTP Requests (using Http4s)
 * Parsing Responses
 * Collections

##What you need
 
You will need the following installed

 * SBT
 * Scala (+Java)
 * Intellij Community Edition (or IDE of your choice)
